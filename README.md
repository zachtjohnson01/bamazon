## Getting Started


### Prerequisites

1. [Node.js](https://nodejs.org/en/)
2. [mySQL](https://dev.mysql.com/doc/)

### Installing

1. Clone the Git repository

   ```
   $ git clone <repo>
   ```
2. Navigate to the directory and install the dependencies 
   ```
   $ npm install
   ```
